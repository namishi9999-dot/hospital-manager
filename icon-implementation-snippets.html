アイコン実装スニペット集

概要

このドキュメントは、Base64エンコードされたアイコン画像をHTMLアプリケーションに統合するための実装スニペット集です。

3つの実装方法を提供します：

  • 方法1： HTMLに直接Base64を組み込み
  • 方法2： 共通スクリプト (icon-base64-loader.js) を使用
  • 方法3： 外部ファイルから動的に読み込み（推奨）

必要なファイル

ファイル名 説明 対象
hospital-icon-base64.txt 病院アイコン (0011) のBase64データ Hospital
converter-icon-base64.txt コンバーターアイコン (0022) のBase64データ Converter
icon-base64-loader.js アイコン読み込みユーティリティスクリプト 両アプリ

方法1: HTMLに直接Base64を組み込み

病院管理アプリ用 (hospital-manager-github.html)

HTMLの <head> タグ内に以下を追加：

<script>
(function() {
  const BASE64 = "iVBORw0KGgoAAAANSUhEUgAA..."; // hospital-icon-base64.txt から取得

  function setIcon() {
    const dataUrl = 'data:image/png;base64,' + BASE64;
    let link = document.querySelector('link[rel="apple-touch-icon"]');
    if (!link) {
      link = document.createElement('link');
      link.rel = 'apple-touch-icon';
      document.head.appendChild(link);
    }
    link.href = dataUrl;
  }

  if (document.readyState === 'loading') {
    document.addEventListener('DOMContentLoaded', setIcon);
  } else {
    setIcon();
  }
})();
</script>

方法3: 外部ファイルから読み込み（推奨）

HTMLの <head> タグ内に以下を追加：

<script src="icon-base64-loader.js" defer></script>
<script>
(async function() {
  if (!window.IconLoader) return;
  const base64 = await window.IconLoader.loadBase64FromFile('hospital-icon-base64.txt');
  if (base64) window.IconLoader.setIcon('hospital', base64);
})();
</script>

検証方法

F12キーでデベロッパーツールを開き、コンソールで以下を実行：

document.querySelector('link[rel="apple-touch-icon"]')
document.querySelector('link[rel="icon"]')

アイコンが表示されていれば成功です。

注意事項

Base64サイズ： 約748KB (hospital) / 837KB (converter)

キャッシュクリア： アイコンが更新されない場合は Ctrl+Shift+Delete でキャッシュをクリアしてください
