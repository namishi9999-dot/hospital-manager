
```

接続テストで確認
```

```
↓

↓
該当施設が表示される
```

### Q: 何度も読む必要はありますか？

### Q: v7の新機能は何ですか？
```
A:
1. 「🏷️ タグ検索」機能追加
2. AND条件フィルタリング
3. タグバッジの非表示化
4. UI全体の改良
```

### Q: 既存機能は変わりましたか？
```
A: いいえ。v1-v6の機能は変更なし。
   v7は新機能の追加とUI改良です。
```

```
A: 大丈夫です。GitHub連携でバックアップされています。
   毎回の変更が自動保存されます。
```
