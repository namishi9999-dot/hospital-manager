## 📖 使い方（3ステップ）

### ステップ1️⃣ : アプリを開く
```
hospital-manager-github.html をブラウザで開く
↓
「📋 病院一覧」タブが表示される
```

### ステップ2️⃣ : 「🏷️ タグ検索」カードをタップ
```
ダッシュボードに以下のカードが表示:
- 📊 すべて（全施設）
- 🗺️ 都道府県別
- 🏷️ タグ検索 ← クリック！
- 📅 外来予定
```

### ステップ3️⃣ : タグをチェックして検索
```
例1: 「初期訪問」だけチェック
  → 「初期訪問」タグを持つ施設が表示

例2: 「初期訪問」と「定期フォロー」の両方チェック
  → 「初期訪問」**かつ**「定期フォロー」両方を持つ施設が表示
    （AND条件で絞り込み）

例3: すべてチェック解除
  → 全施設が表示
```

## 💡 Tips

### Tip 1️⃣ : AND検索とは？
```
「初期訪問」AND「定期フォロー」
= 両方のタグを持つ施設のみ表示

例:
施設A: 「初期訪問」「定期フォロー」「研修サポート」
施設B: 「初期訪問」のみ
施設C: 「定期フォロー」「研修サポート」

「初期訪問」と「定期フォロー」をチェック
→ 施設Aだけ表示される
```

### Tip 2️⃣ : 検索をリセットする
```
全てのチェックを外す
↓
すべての施設が表示される
```

### Tip 3️⃣ : 詳細情報を確認する
```
施設カードをタップ
↓
すべての情報が表示される
- 施設名、住所、電話
- 病床数、ICU病床数、透析室病床数
- 透析患者数、透析装置更新年度
- メモ、タグ
```

## ❓ よくある質問

### Q1: タグを新規作成するには？
```
⚙️ 設定タブ → 「新規ターゲット(タグ)追加」セクション
タグ名を入力して追加ボタンをクリック
```

### Q2: 施設にタグを付与するには？
```
施設カードをタップ → 「編集」ボタン
下部の「ターゲット(タグ)」セクションでチェック
保存ボタンをクリック
```

### Q3: 複数の施設に同じタグを付けたい
```
各施設を編集して、同じタグをチェック
（個別に追加する必要があります）
```

### Q4: タグを削除するには？
```
⚙️ 設定タブ → 「新規ターゲット(タグ)追加」セクション
削除したいタグの「×」ボタンをクリック
```

### Q5: タグバッジが見たい
```
施設カードをタップして詳細情報を表示
→ 詳細モーダルで「タグ」セクションを確認
```

## 🎯 使用シーン別ガイド

### シーン1: 「初期訪問が必要な施設」を確認したい
```
1. 「🏷️ タグ検索」カードをタップ
2. 「初期訪問」チェックボックスをチェック
3. 該当施設が一覧表示
4. 施設をタップして詳細・電話番号を確認
```

### シーン2: 「初期訪問も定期フォローも必要な施設」を確認したい
```
1. 「🏷️ タグ検索」カードをタップ
2. 「初期訪問」と「定期フォロー」の両方をチェック
3. 両方のタグを持つ施設のみ表示
4. リスト化して営業計画に活用
```

### シーン3: 「都道府県別で見たい」場合
```
1. 「🗺️ 都道府県別」カードをタップ（従来通り）
2. 都道府県ごとに施設が グループ化表示される
```

### シーン4: 「外来スケジュールを登録・確認したい」場合
```
1. 「📅 外来予定」カードをタップ（従来通り）

```

## 🆘 トラブル対応

### 問題: タグ検索で施設が表示されない
```
✅ 対策1: 施設にタグが付与されているか確認
  → 施設編集画面でタグをチェック

✅ 対策2: 複数タグが選択されていないか確認
  → チェックボックスを1つだけ選択して試す

✅ 対策3: キャッシュをクリア
  → ブラウザをリロード（F5キー）
  → ⚙️ 設定タブの「更新」をクリック
```

### 問題: 変更がGitHubに保存されない
```
✅ 対策1: GitHub設定を確認
  → ⚙️ 設定タブで「接続をテスト」をクリック

✅ 対策2: インターネット接続を確認
  → ブラウザのコンソール (F12) でエラーを確認

✅ 対策3: Personal Access Token が有効か確認
  → https://github.com/settings/tokens から確認
```

### 問題: 病院カードにまだタグバッジが表示されている
```
✅ 対策: キャッシュをクリア
  → ブラウザをリロード（Ctrl+F5 または Cmd+Shift+R）
  → hospital-manager-github.html を再度開く
```

## 📞 サポート情報

### よくあるエラーとその対応

| エラー | 原因 | 対策 |
|--------|------|------|
| 「データを読み込めません」 | GitHub接続失敗 | 設定を確認、トークンを確認 |
| 「該当する施設がありません」 | タグが施設に付与されていない | 施設編集でタグをチェック |
| データが古い | キャッシュが残っている | ⚙️ 更新ボタンをクリック |
| 複数タグで検索できない | AND条件を理解していない | 上記「Tip 1️⃣」を参照 |
