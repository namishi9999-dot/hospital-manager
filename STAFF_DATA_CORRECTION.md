
## 注意事項

### メモ列について

Memo列には診療科・部門情報が複数含まれる場合があります：
```
Specialty: 循環器内科
Specialty: 消化器外科 | Department: 診療科A
```

- **Specialty**: 診療科や専門分野
- **Department**: より詳細な部門情報

### 名前に空白がある場合

一部のデータでは名前と苗字の間に複数スペースが存在します（Excelの仕様）。
```
「宮崎  俊明」（空白2つ）
「岸良  幸」（空白2つ）
```

これは修正済みで、Excelからの直接マッピングとなっています。
